
Lancer le script avec la commande:
```sh run.sh```

Le script aura besoin d’accès à internet pour télécharger les modèles entraînés et les librairies nécessaires.
Une fois le serveur web Flask lancé, vous pouvez y accéder sur n’importe quelle navigateur grâce à l’url suivant: localhost:5000. 
Ici le port par défaut sera 5000 si ce n’est pas le cas l’url sera affiché sur le terminal.
